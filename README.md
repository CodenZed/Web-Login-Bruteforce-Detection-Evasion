# GitHub BruteForceAI 🚀

Advanced GitHub login brute-force & password spraying tool for **authorized penetration testing**.

## Features
- 🎭 Playwright browser automation with headless Chromium
- 🧠 LLM-style login form analysis & DOM diff success detection
- ⚡ Async multi-threading (configurable)
- 📊 SQLite results database with credential hashing
- 🔔 Discord/Slack webhook notifications
- 🌐 Proxy support (SOCKS5/HTTP)
- 📈 Two attack modes: `bruteforce` & `spray`
